# ESP32 kit "tinkering"

This repository contains the projects, experiments, and random code I'm writing while learning and playing around with my ESP32 development kit.

I recently picked up an ESP32 kit and decided to upload the different things I'm experimenting with as I learn. This isn't intended to be a polished project, it's more of a personal playground where I can try out ideas, test hardware, and learn how everything works.

Expect to find things like:

- LED and GPIO experiments
- Wi-Fi and Bluetooth examples
- Sensor integrations
- Display projects
- Home automation ideas
- Random prototypes and proof-of-concepts
- Anything else I feel like tinkering with
